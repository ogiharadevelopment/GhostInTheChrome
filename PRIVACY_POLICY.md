# プライバシーポリシー・利用規約

## プライバシーポリシー

### 1. 収集する情報

Ghost In The Chrome拡張機能は、以下の情報を収集・保存します：

#### 1.1 ローカルストレージに保存される情報
- **ブックマーク情報**: ユーザーのブックマーク（タイトル、URL、追加日時）
- **お気に入り情報**: ユーザーが追加したお気に入り（タイトル、URL、追加日時）
- **設定情報**: GUIの位置設定、履歴表示数設定
- **ゴーストモード設定**: 通常/ステルス/無効モードの設定

#### 1.2 課金関連情報
- **課金状態**: Chrome Web Store Payments APIを通じて取得される課金状態
- **暗号化された課金情報**: セキュリティのため暗号化して保存

### 2. 情報の使用目的

収集した情報は以下の目的でのみ使用されます：

- **機能提供**: ブックマーク・履歴・お気に入りの表示
- **設定の保存**: ユーザーの設定を次回起動時に復元
- **課金機能**: プレミアム機能の有効/無効制御

### 3. 情報の共有

- **外部への送信**: 収集した情報を外部に送信することはありません
- **第三者提供**: ユーザーの同意なしに第三者に情報を提供することはありません
- **統計情報**: 匿名化された統計情報のみが開発改善に使用されます

### 4. データの削除

- **手動削除**: 拡張機能のアンインストール時にすべてのデータが削除されます
- **設定リセット**: 設定画面から個別にデータを削除可能です

### 5. セキュリティ

- **暗号化**: 機密情報はAES-GCM暗号化で保護されます
- **ローカル保存**: すべてのデータはユーザーのデバイス内に保存されます
- **アクセス制限**: 拡張機能以外からのデータアクセスを制限します

## 利用規約

### 1. 利用条件

本拡張機能の利用には以下の条件を満たす必要があります：

- **Chromeブラウザ**: Google Chrome 88.0以上
- **権限同意**: 必要な権限（ブックマーク、履歴、ストレージ）への同意
- **適法利用**: 法律に違反する目的での使用禁止

### 2. 無料機能

#### 2.1 基本機能
- **ブックマーク表示**: ユーザーのブックマークを表示
- **履歴表示**: 最近の履歴を表示（最大150件）
- **透明GUI**: 画面右端の透明なインターフェース
- **スクロール操作**: マウスホイールでの操作

#### 2.2 制限事項
- **履歴表示数**: 無料版は最大150件まで
- **お気に入り機能**: 利用不可
- **カスタム設定**: 履歴表示数の変更不可

### 3. プレミアム機能（課金）

#### 3.1 追加機能
- **お気に入り機能**: ブックマークをお気に入りに追加・管理
- **カスタム履歴**: 履歴表示数を300-1000件にカスタマイズ
- **高度な検索**: 履歴・ブックマークの高度な検索機能

#### 3.2 課金について
- **課金方法**: Chrome Web Store Payments API
- **課金状態**: 暗号化して安全に保存
- **課金確認**: 定期的に課金状態を確認

### 4. 禁止事項

以下の行為は禁止されています：

- **逆コンパイル**: 拡張機能の逆コンパイル・解析
- **改変**: 拡張機能の改変・再配布
- **悪用**: 他のユーザーに迷惑をかける使用
- **違法利用**: 法律に違反する目的での使用

### 5. 免責事項

#### 5.1 機能保証
- **可用性**: 機能の完全性・正確性を保証しません
- **継続性**: サービスの継続提供を保証しません
- **互換性**: 将来のChromeアップデートとの互換性を保証しません

#### 5.2 損害賠償
- **間接損害**: 拡張機能の使用による間接損害について責任を負いません
- **データ損失**: データの損失・破損について責任を負いません

### 6. 変更・終了

#### 6.1 規約変更
- **事前通知**: 重要な変更は事前に通知します
- **継続利用**: 変更後も利用を継続した場合、新しい規約に同意したものとみなします

#### 6.2 サービス終了
- **事前通知**: サービス終了時は事前に通知します
- **データ削除**: 終了時にすべてのデータを削除します
